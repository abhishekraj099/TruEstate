# Frontend - TruEstate Sales Management System

React-based user interface for sales management with real-time filtering and search.

## Components

- **SearchBar**: Real-time search input
- **FilterPanel**: Multi-criteria filter sidebar
- **SalesTable**: Data table with formatted columns
- **SortDropdown**: Sorting options selector
- **Pagination**: Navigation controls

## Setup


Runs on `http://localhost:3000`

## Build


Production build output in `dist/` folder.
